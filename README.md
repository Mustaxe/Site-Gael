GAEL
===============

Requisitos
----------
 - [NodeJS]
 - [GruntJS]

[NodeJS]: http://nodejs.org/
[GruntJS]: http://gruntjs.com/

Instalação
----------

 - Instalar os modulos do NodeJs para o build da aplicação

 ```
 npm install
 ```

Automação de tarefas
--------------------


Build da aplicação
------------------

- Minificar os css, svg, js e concatenar os vendors.

 ```
 grunt build
 ```
